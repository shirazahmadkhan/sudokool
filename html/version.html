Sudokool V 2.5 – 15 Feb 2009

  • Fully controllable through keyboard

  • Auto Submit instead of manual

  • Fastest timings stored with username

  • Warning made dynamic and stored as a preference

  • Storing user statistics

Sudokool V 2.0 – 12 Oct 2008

  • Puzzle generator added to generate unlimited puzzles

  • Improved puzzle solver algorithm

  • 4 difficulty levels : simple, easy, medium and hard

  • Hints added : show all the possibilities on the puzzle board

  • GUI improvements

  • Rating and lowest time recording for custom puzzles also

  • fixed some old bugs

Sudokool v 1.0 – 25 Sep 2005

  • 300 puzzles in 3 difficulty levels : easy, medium and hard

  • user can play custom puzzles

  • games can be saved and loaded

  • warning for wrong answer can be toggled

  • user can pause/resume game

  • 3 lowest records for each difficulty level
