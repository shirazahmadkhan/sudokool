Sudokool can be fully controlled through keyboard.
Each top menu icon has an underlined alphabet in the label name and can be controlled using ALT key with this underlined alphabet. e.g.

New : ALT + N to play a new game
Load : ALT + O to load a saved game
Save : ALT +V to save a game
Custom : ALT + C to play a custom game
Pause : ALT + P to Pause game
Hints : ALT + I to turn Hints ON/OFF
Warn : ALT +W to turn Warnings ON/OFF
Again: ALT + A to play same puzzle again
Solution: ALT + S to view the solution of the puzzle
Records : ALT + R to view the game statistics
Help : ALT +H to open help window
Exit : ALT +X to exit Sudokool

Use arrow keys to move over puzzle grid.
Pressing a number key over a blank button makes the entry.
Use ALT + number combination for making a pencil entry
Use Del key to delete an entry
